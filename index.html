Speech Recognition
Text color indicates how confident the speech recognition system is, that the recognition is correct:

0 - 40%

40% - 60%

60% - 80%

80% - 100%

The API can listen to certain words / sentences and react accordingly (eg. trigger an action).

For example it can change the transcripted text, and replace it with something else... or call the NSA, ABW or FSB. Just say "explosives", "bomb", "weapon". You can say "cat" or "kitten" too. Also in Polish.

Another example: it will change the color of this paragraph and triggers some animation when hears color's name (for the purpose of this example only a limited number of colors (English), the main ones, like blue, red, yellow etc.)
